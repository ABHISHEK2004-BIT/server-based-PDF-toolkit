## 📌 Overview

**PDF Toolkit Pro** is a server-based web application that allows users to perform various PDF operations such as merging, splitting, compressing, and managing PDF files through a user-friendly interface.

The application is designed to support **multi-user access over LAN**, making it ideal for office environments.

---

## 🚀 Features

- 📎 Merge multiple PDF files
- ✂ Split PDF into pages
- 🗜 Compress PDF files
- 🔄 Rearrange pages
- 📤 Upload & download PDFs
- 🌐 Multi-user access via server (LAN-based)
- ⚡ Fast processing with Python backend
